This is a simple app demonstarting the twitter stream visualization.

It will generate tag cloud of most frequently used words, most frequent used hashtags and most frequent users whom you mentioned in your tweets.
It will also generate bar, pie and donut charts of tweets daywise, weekdays-wise, hour-wise and source of your tweets.

The app uses Python 2.7, jQuery 1.7, jQPlot and jCloud plugins.
The source code of this app is available on my GitHub account.

The app will run only on Firefox, Safari and Chrome.
Internet Explorer doesnot supports the plugins used in this app.
